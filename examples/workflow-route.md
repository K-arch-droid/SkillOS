# 示例：路由用户请求到合适 Skill

## 场景

你不确定某个任务应该用哪个 Skill。

## 命令

```bash
python skillos.py route "帮我写单元测试"
```

## 输出示例

```
# Skill 路由结果

**用户请求：** 帮我写单元测试

## 推荐 Skill

| 排名 | Skill                     | 置信度 | 原因                    |
|------|---------------------------|--------|-------------------------|
| 1    | javascript-testing-patterns | 72%  | 领域匹配；关键词重叠 50% |
| 2    | webapp-testing             | 35%   | 关键词重叠 30%          |

## 最佳匹配：javascript-testing-patterns

- 置信度：72%
- 原因：领域匹配；关键词重叠 50%
```

## 路由逻辑

1. **意图分析** — 识别任务类型、领域
2. **关键词匹配** — 与 Skill description 对比
3. **语义匹配** — 领域关键词映射
4. **置信度排序** — 返回 Top 5

## 下一步

根据推荐安装或直接使用对应 Skill。
